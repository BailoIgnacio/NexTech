CAMBIOS REALIZADOS AL PROYECTO BASE
-------------------------------------
- Backend migrado de Node.js a Python + FastAPI (requisito obligatorio del TP)
- Persistencia via archivos JSON en backend/data/ (sin base de datos)
- Sistema de autenticacion: registro, login usuario, login admin
- Usuarios precargados en backend/data/users.json
- Logs de aplicacion en backend/logs/app.log
- Tests de ejemplo en backend/tests/ (pytest)
- Frontend HTML/CSS/JS sin cambios (se sirve desde FastAPI)

COMO CORRER EL PROYECTO (nuevo)
---------------------------------
  1. Tener Python 3.10+ instalado
  2. Abrir terminal en la carpeta backend/:
       cd backend
  3. Instalar dependencias:
       pip install -r requirements.txt
  4. Iniciar el servidor:
        python -m uvicorn main:app --reload --port 3000 
  5. Abrir en el navegador:
       http://localhost:3000/
  6. Documentacion automatica de la API (Swagger):
       http://localhost:3000/docs

ENDPOINTS DISPONIBLES
-----------------------
  GET    /api/products              Lista productos con filtros y paginacion
  GET    /api/products/:id          Obtener un producto por ID
  POST   /api/products              Crear producto
  PUT    /api/products/:id          Reemplazar producto
  PATCH  /api/products/:id          Actualizar parcialmente
  DELETE /api/products/:id          Eliminar producto
  GET    /api/stats                 Estadisticas generales
  POST   /api/auth/register         Registrar usuario nuevo
  POST   /api/auth/login            Login (usuario comun o admin)
